# USMLE Notes Project

This project contains a collection of notes and knowledge bases related to the USMLE (United States Medical Licensing Examination).

## Files

* `knowledge_base_complete.json`: A JSON file containing a comprehensive knowledge base of USMLE notes.
* `knowledge_map_v9.html`: An HTML file that visualizes the knowledge base as a map.

## Usage

To use this project, you can:

1. Browse the `knowledge_map_v9.html` file in your web browser to explore the knowledge map.
2. Use the `knowledge_base_complete.json` file as a data source for your own applications or scripts.
